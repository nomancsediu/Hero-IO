## Features

- Browse and search through a curated catalog of apps
- View detailed app info including ratings and download stats
- Install and uninstall apps with localStorage persistence
- Sort installed apps by size
- Loading animations on page transitions
- Custom 404 error page
- Fully responsive for all screen sizes

## Tech Stack

- **React 19**
- **Vite**
- **React Router v7**
- **Tailwind CSS v4**
- **DaisyUI**
- **SweetAlert2**
- **react-hot-toast**
- **Font Awesome**
- **Google Fonts (Inter)**
